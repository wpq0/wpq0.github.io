It was Bkav's entry in GDT's initiative to bring in the private sector called TVAN (similar to e-file model by the IRS in America). Though being the last to come in the race, my team's effort made Bkav the 2nd company to be granted GDT's license, even before bigger players like Viettel and FPT. Our service is also the major factor behind the sales boost of BkavCA's certificate USB token.
    Details »

    • Laid out the overall architecture and built the common infrastructure libraries.
    • Define a hyper schema so that data with constantly changing schema can be effectively stored in a relational database. This was crucial because GDT update fields and rules on their tax forms very frequently.
    • Builted an simple formula engine (with similar syntax to Excel) in which domain experts can define complex rules. The AST gained from parsing these formulas was used to:
      • Weave JavaScript code to feed the browser (client-side)
      • Evaluate them on .NET's DLR (server-side)
      • Reduce by 50% the work on auto-correction and validation of tax forms by uniting the code on both sides. This is the major time saver in the course of our project, since there is a huge number of complex mathematical formulas in the tax forms.
    • Design an web interface to create and modify tax form schemas so the burden of creating 100+ tax forms is shifted from the developers to the domain experts.
    • Put together a library for exporting user's tax form data into valid PDF and Excel documents.
    • Wrote a C♯ library and a Java applet to stamp digital signature on the exported documents both in the browser and on the server.
    • Created a WPF desktop application so those without Java can download and sign PDF tax returns on their own machine.
    • Hancrafted the networking component that talks to a legacy SwA web service on GDT's side. (SwA was proposed by W3C decades ago, but never became a standard).
    • Held regular code review sessions and trained other team members in the use of design patterns, unit testing and issue tracking.
