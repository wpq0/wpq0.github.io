Bkav TVAN

Type
Web application
Company
BKAV
Url
noptokhai.vn
Role
Team leader, lead developer

This was the company's bid in the GDT's initiative to create a model similar to the IRS's e-file system in the U.S. My team's effort helped the company secure the license before bigger players like Viettel and FPT. Our service also boosts the sale of BkavCA's certificate tokens.

I was responsible for all the core components:

  • A form-builder where editors can define the schema of all tax returns. With this, the burden of creating and maintaining 100+ tax forms was shifted from the developers to domain experts.
  • An expression parser that let them define complex formulae in those forms using a syntax similar to Excel. This parser produce an AST that we then use to generate Javascript to validate input in the browser as well as to evaluate form values on the DLR server-side.
  • A library for generating, signing and validating digital signatures on tax returns in PDF and Excel formats. It was so versatile that I could use it on the server, put it in our WPF client app, and translated it to a Java applet.
